## Project Overview

This website provides information about Zero Associate's trading, contracting, and maintenance services in Qatar. The revamp focused on:

- **Enhanced User Experience:** Improved navigation, mobile responsiveness, and interactive elements like a chatbot.
- **Unified Service Page Layouts:** A consistent and professional design for all service detail pages.
- **Robust Contact Form:** Integration with Google Sheets for reliable form submissions.
- **Accessibility & SEO Improvements:** Better HTML semantics, ARIA attributes, and image alt tags.
- **Code Cleanliness:** Organized and commented code for easier maintenance.
